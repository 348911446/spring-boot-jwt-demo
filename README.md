方案A：使用预编译Ghidra（⭐ 强烈推荐）
这是最简单且最实用的方案：

步骤：

在有网络的电脑上访问：https://github.com/NationalSecurityAgency/ghidra/releases

下载最新版本的 ghidra_XX.X_public_YYYYMMDD.zip 文件（约700MB）

例如：ghidra_10.4_public_20250201.zip
复制到您的离线电脑（用USB或其他方式）

在离线电脑上解压并运行：


ghidra\bin\ghidra.exe
优点：

✅ 完整的功能，包括Decompiler
✅ 无需编译，立即可用
✅ 官方预编译，性能最优
